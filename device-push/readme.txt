== Installation ==

= From within WordPress =
1. Visit 'Plugins > Add New'
2. Search for 'Device Push'
3. Activate Device Push from your Plugins page.
4. Go to "after activation" below.

= Manually =
1. Upload the "device-push" folder to the "/wp-content/plugins/" directory
2. Activate the Device Push plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.

= After activation =
1. Active and configure your Device Push User Account
2. Create into adminitration panel of Device Push your first application/web
3. Active when you want send your push notifications
